# API-Automation-Framework

##Tech Stack
-Java
-RestAssured
-TestNG
-Maven
-Extent Reports

#Framework Structure 
-pojo/ -Request Response Models
-utils/ -BaseTest and ReportManager
-tests/ -Test classes

     
##Test Cases
-Login API
-Create Product
-Create Order
-View Order Details
-Delete Product

##How to Run
1. Clone Repo
2. Open in Eclipse as Mave Project
3. Right-click->Run As->TestNG Test

##API Under Test
-Base URL : https://rahulshettyacademy.com
-Auth : JWT Token

##Reports
Extent HTML report generated in reports/folder after execution
